🌻HTML

🌸 meta tag

Role: set metadata for HTML site.

Note: meta tag is a void element. That means the begin tag must be presentand the end tag must not be present.

Attributes:
   1. charset: Set encoded type for the site. Recommended to use utf-8.
   2. content: Contain the value for html-equiv or name attributes.
   3. html-equiv: Define pragma directive (content-security-policy, refresh, set-cookie)
      Ex: <meta http-equiv="refresh" content="3;url=https://www.mozilla.org">
   4. name: Define document info like description, author...
      Ex: <meta name="description" content="This is a website developed by NhungBH">      
The content above will be shown on search result of Google

🌸 Format text

To italize: Recommended to use <em> instead of <i>
To bold: Recommended to use <strong> instead of <b>
The results do not look different but <em> and <strong> tell brower that the text is important
